# SynthSemi Website

A modern, responsive, SEO-optimized single-page React.js website for SynthSemi - ASIC IP and Design Services Company.

## Features

- ⚡ Fast and optimized with Vite
- 📱 Fully responsive design
- 🔍 SEO optimized with React Helmet
- 🎨 Modern and clean UI
- ♿ Accessible design
- 🚀 Production-ready build

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 18
- Vite
- React Helmet Async (SEO)
- Modern CSS with Flexbox/Grid

## SEO Features

- Semantic HTML structure
- Meta tags optimization
- Open Graph tags
- Twitter Card tags
- Canonical URLs
- Structured data ready
